# Controller Projesi

Bu proje, Python tabanlı bir sunucu uygulamasıdır. Kullanıcıdan gelen istekleri işleyerek belirli işlemleri gerçekleştirmek üzere tasarlanmıştır.

## Özellikler

- Python ile geliştirilmiş sunucu uygulaması
- Kolay kurulum ve kullanım
- Geliştirilmeye açık, modüler yapı

## Kurulum

1. **Python Yükleyin:**

   - Proje Python ile yazılmıştır. [Python'un resmi web sitesinden](https://www.python.org/downloads/) Python 3.8 veya üzeri bir sürümü indirip kurun.

2. **Projeyi İndirin:**

   - Bu repository'yi bilgisayarınıza klonlayın veya ZIP olarak indirip çıkarın.

## Kullanım

1. Komut satırında proje klasörüne gidin:
   ```cmd
   cd "c:\Users\recep\Desktop\Beykoz Edu\controller-receperturkk"
   ```
2. Sunucuyu başlatmak için:
   ```cmd
   python server.py
   ```
3. Sunucu çalışmaya başladıktan sonra, istemci uygulamanız veya tarayıcı üzerinden sunucuya istek gönderebilirsiniz.
